### Why these files are GPL-3.0

The generated Go bindings inherit the license of their `.proto` inputs
under the Protocol Buffers license, which states:

> Code generated by the Protocol Buffer compiler is owned by the owner
> of the input file used when generating it. This code is not standalone
> and requires a support library to be linked with it. This support
> library is itself covered by the above license.

The input `.proto` files are owned by the Meshtastic project and
distributed under GPL-3.0, so the generated Go bindings in
`internal/radio/meshtasticpb/` inherit that license. They are kept in a
dedicated subdirectory and do not affect the license of MeshGo's
original source code.

### Distribution

MeshGo bundles these generated files in its desktop binaries. The
complete corresponding source is available in this repository under
`internal/radio/meshtasticpb/`, which satisfies GPL-3.0's
source-availability requirement for binary distribution.
